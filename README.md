## 🚀 핵심 기술
### 1. 병렬 라우팅
#### - @modal 슬롯을 활용한 병렬 라우트 구현
#### - 페이지 상태 유지하면서 모달 오버레이
#### - 뒷 페이지 유지 + 모달 동시 렌더링

### 2. 인터셉트 라우팅
#### - (.)info/[id] 패턴을 사용한 라우트 인터셉트
#### - 같은 페이지에서 모달로 포켓몬 상세 정보 표시
#### - URL 변경과 함께 모달 상태 관리

### 3. 스켈레톤 UI
#### - Suspense fallback을 활용한 비동기 렌더링 처리
#### - 포켓몬 리스트 로딩 시, 스켈레톤 UI를 표시하여 사용자 이탈 방지
<br />

### Next Image 컴포넌트 활용
